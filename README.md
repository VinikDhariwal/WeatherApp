# WeatherApp

A responsive weather search app built with React and Material UI. Search any city in the world and get current weather conditions instantly.

## Features

- Current temperature, min, max, and feels like
- Humidity, wind speed, visibility, and pressure
- Cloud coverage, sunrise, and sunset times
- Weather icon from OpenWeatherMap
- Error handling for invalid city names
- Clean responsive UI with Material UI components

## Tech Stack

- React (Vite)
- Material UI
- OpenWeatherMap API
- CSS
- MERN
